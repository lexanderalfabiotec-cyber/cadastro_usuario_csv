## Why

A necessidade de uma interface simples para capturar e registrar dados básicos de usuários (nome e idade). Este aplicativo resolve o problema de coleta de dados rápida sem a complexidade de um banco de dados backend, permitindo que os usuários salvem as informações localmente em formato CSV.

## What Changes

- Implementação de uma aplicação React moderna.
- Criação de um formulário de cadastro com validação de campos.
- Desenvolvimento de um sistema de persistência local temporária (estado da aplicação) e exportação para CSV.
- Interface responsiva com foco em usabilidade.

## Capabilities

### New Capabilities
- `registration-form`: Interface de formulário para entrada de Nome e Idade, incluindo tratamento de eventos e feedback visual.
- `csv-storage`: Mecanismo para converter os dados coletados em strings formatadas em CSV e possibilitar o download do arquivo pelo navegador.
